📝 LangChain 示例代码

1. 环境配置和依赖安装

# 安装 LangChain 及相关依赖
pip install langchain langchain-openai langchain-anthropic
pip install langchain-core langchain-community

# 设置环境变量
export OPENAI_API_KEY="your-api-key-here"
export ANTHROPIC_API_KEY="your-api-key-here"
# Node.js 环境配置
npm install langchain langchain-openai

import { ChatOpenAI } from "langchain/chat_models/openai";

2. 构建对话机器人 (Python)

from langchain_openai import ChatOpenAI
from langchain_core.prompts import ChatPromptTemplate
from langchain_core.output_parsers import StrOutputParser
from langchain_core.runnables import RunnablePassthrough

# 初始化 ChatOpenAI 模型
llm = ChatOpenAI(
    model="gpt-4",
    temperature=0.7,
    streaming=True  # 启用流式输出
)

# 创建提示词模板
prompt = ChatPromptTemplate.from_messages([
    ("system", "你是一位专业的 {topic} 导师，用简洁易懂的语言解释概念。"),
    ("human", "{question}")
])

# 使用 LCEL 串联组件
chain = prompt | llm | StrOutputParser()

# 执行问答
response = chain.invoke({
    "topic": "人工智能",
    "question": "什么是 LangChain？"
})

print(response)

3. 流式输出实现

# 同步流式输出
for chunk in chain.stream({
    "topic": "编程",
    "question": "解释一下什么是回调函数"
}):
    print(chunk, end="", flush=True)

# 异步流式输出
async def stream_response():
    async for chunk in chain.astream({
        "topic": "Web开发",
        "question": "什么是 RESTful API？"
    }):
        print(chunk, end="", flush=True)

# 带回调的流式处理
from langchain.callbacks.streaming_stdout import StreamingStdOutCallbackHandler

llm_with_screening = ChatOpenAI(
    model="gpt-4",
    callbacks=[StreamingStdOutCallbackHandler()]
)

# LCEL 链式调用
result = (prompt | llm_with_screening).invoke({
    "topic": "数据库",
    "question": "什么是 ACID 属性？"
})

📖 README 文档

项目介绍

这是一个基于 LangChain 构建的智能对话机器人示例项目。本项目演示了如何使用 LangChain Expression Language (LCEL) 构建完整的问答系统。

核心特性

  • ✨ LCEL 链式调用 - 使用管道操作符串联 Prompt、Model、OutputParser
  • 🌊 流式输出 - 支持实时流式响应，提升用户体验
  • 🎨 现代化 UI - Glassmorphism 风格的深色主题界面
  • 📚 完整文档 - 包含代码示例和使用说明

技术栈

  • 后端: Python, LangChain, OpenAI API / Anthropic API
  • 前端: Pure HTML/CSS/JavaScript
  • UI 设计: Glassmorphism Dark Theme

快速开始

  1. 安装 Python 和必要的依赖包
  2. 配置 OpenAI API Key
  3. 运行后端服务
  4. 打开前端页面开始对话

LCEL 核心概念

LangChain Expression Language (LCEL) 是一种声明式的方式来组合 LangChain 组件。核心特性包括：

  • | 管道操作符: 将上一个组件的输出传递给下一个组件
  • Runnable 接口: 所有组件都实现统一的 Runnable 接口
  • 流式支持: 原生支持流式处理和异步操作
  • 批处理: 内置支持批量处理和并行执行
